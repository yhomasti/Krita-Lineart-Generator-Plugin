LineArt-O-Matic

Converts sketches to clean line art with optional SVG export.

Usage

  1. Open a sketch in Krita
  2. Select the layer to process
  3. Go to Tools ‚Üí Scripts ‚Üí LineArt-O-Matic
  4. Result appears as new layer + optional SVG export

Requirements

OpenCV must be installed in Krita's Python environment.
